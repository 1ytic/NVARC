<rules_summary>
For each non-zero number in the input, draw a border around the output grid with that number. The location of the original number determines the row/column where the border starts.
</rules_summary>
<input_generation>
Input grids of varying sizes are provided. The transformation maintains relationships between input and output dimensions.
</input_generation>
<solution_steps>
1. Examine the input grid for non-zero numbers.
2. For each non-zero number, draw a filled border in the output grid. The border starts at the row/column of the non-zero number in the input grid.
3. Fill the border area with the non-zero number.
</solution_steps>
<key_insight>
The grid size remains constant.
</key_insight>
<puzzle_concepts>
- boundary
- filling
- tiling
</puzzle_concepts>
