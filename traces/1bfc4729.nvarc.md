<rules_summary>
The non-zero value in the input is propagated to the top and bottom rows, as well as the left and right columns, of the output grid.
</rules_summary>
<input_generation>
Input grids of varying sizes are provided. The transformation maintains relationships between input and output dimensions.
</input_generation>
<solution_steps>
1. Find the non-zero value in the input grid.
2. Find the indices of the non-zero value.
3. Copy the input grid to the output grid.
4. Set the top and bottom rows of the output grid to the non-zero value.
5. Set the left and right columns of the output grid to the non-zero value.
</solution_steps>
<key_insight>
The input grid contains a single non-zero value.
</key_insight>
<puzzle_concepts>
- boundary
- filling
</puzzle_concepts>
