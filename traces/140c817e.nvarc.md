<rules_summary>
The output grid has the same dimensions as the input. The location of the non-background color in the input is used as a central point. The original non-background color is duplicated to form a cross. New colors 2 and 3 fill in spaces in the output.
</rules_summary>
<input_generation>
Input grids contain colored cells. The transformation operates on color patterns and relationships.
</input_generation>
<solution_steps>
1. Identify the location of the non-background color in the input grid.
2. Create an output grid with the same dimensions as the input grid, filled with the background color.
3. Place the non-background color at the same location as the input grid.
4. Fill the locations horizontally and vertically with color 1.
5. Place color 2 on the horizontal and vertical axis from color 1
6. Fill regions near color 1 and color 2 with color 3.
</solution_steps>
<key_insight>
Input grids contain a single non-background color.
</key_insight>
