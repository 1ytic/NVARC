<rules_summary>
Move all non-zero elements to the top of the grid, maintaining their relative horizontal positions, and filling the remaining space with zeros.
</rules_summary>
<input_generation>
Input grids contain colored cells. The transformation operates on color patterns and relationships.
</input_generation>
<solution_steps>
1. Identify all non-zero elements in the input grid.
2. Preserve the horizontal positions of these elements.
3. Place the rows containing these non-zero elements at the top of the output grid.
4. Fill the remaining rows of the output grid with zeros.
</solution_steps>
<key_insight>
Objects in the input move to the top of the output grid.
</key_insight>
