</rules_summary>
<input_generation>
Input grids of varying sizes are provided. The transformation maintains relationships between input and output dimensions.
</input_generation>
<solution_steps>

2. If a cell has a non-zero value, fill all cells in the corresponding row and column of the output grid with that value.
</solution_steps>
<key_insight>
The output grid has the same dimensions as the input grid.
